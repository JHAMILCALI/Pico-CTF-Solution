## ¿Qué es PicoCTF?

**PicoCTF** es una competencia de ciberseguridad gratuita creada por la **Carnegie Mellon University**, diseñada especialmente para estudiantes y personas que están comenzando en el mundo de la seguridad informática. Los retos están organizados por categorías como criptografía, forense, ingeniería inversa, explotación binaria y más.

Este repositorio documenta mi recorrido personal resolviendo los desafíos de **The Beginner's Guide to the picoGym**, con todas las soluciones explicadas paso a paso **en español**.

---

## Objetivo

> Aprender los fundamentos de la ciberseguridad de manera práctica, documentando cada solución para que sirva de referencia tanto a mí como a otros hispanohablantes que estén comenzando en CTF.

---

## Estructura del Repositorio

```
Pico CTF/
└── The Beginner's Guide to the picoGym/
    ├── Section 1 (Sanity)/          # Retos introductorios
    ├── Section 2 (CyberChef)/       # Retos con herramientas online
    └── Section 3/                   # Comandos y herramientas de terminal
```

Cada reto tiene su propia carpeta con un archivo `Respuesta.md` que contiene:
- La descripción del problema
- El proceso de resolución paso a paso
- Los comandos o herramientas utilizadas
- La flag encontrada

## Formato de una Flag

Todas las flags de PicoCTF siguen este formato:

```
picoCTF{texto_de_la_flag}
```

## Cómo usar este repositorio

1. Navega a la sección que te interese.
2. Abre el archivo `Respuesta.md` del reto.
3. Lee la explicación paso a paso antes de ver la flag.
4. ¡Intenta resolverlo tú mismo primero!
